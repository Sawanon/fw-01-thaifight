มวยพักยก

โปรดอ่านและทำความเข้าใจระเบียบการเล่นนี้ เพื่อประโยชน์ของตัวท่านเอง

1.กรณีมีมวยที่ถูกไล่ลงถือเป็น โมฆะ (ผลมวยในเกมส์)

2.กรณีมวยสลับมุมถือว่าได้เสียตามปกติ แต่จะทำการยึดตามชื่อนักมวยแทน

3.กรณีขึ้นชื่อมวยผิดยกเลิกไม่ได้เสีย

4.กรณีเปิดราคาผิดเกิน 30% ตามท้องตลาดขอสงวนสิทธ์ในการยกเลิกไม่ได้เสีย

5.เปิดราคาผิดมุม (สลับฝั่ง) ขอสงวนสิทธ์ในการยกเลิก ไม่ได้เสีย

6.ผลการชกทางเว็บจะตัดสินยึดตามกรรมการชูมือบนเวที (เมื่อนักมวยทั้งสองลงจากเวที ถือว่าผลเป็นอันสิ้นสุด)

PAKYOK Point

แต้มสะสม PAKYOK Point ใช้ทำอะไร ?

สมาชิกสามารถนำ PAKYOK Point ที่ได้รับไปแลกการ์ด ในเมนู การ์ดนำโชค โดยจะได้รับรางวัลเป็น เครดิตเดิมพัน

วิธีใช้แต้มPAKYOK Point แลกของรางวัล

1.เลือกเมนู รางวัล (Reward)

2.เลือกรางวัลที่ต้องการแลก ประกอบด้วย การ์ดนำโชค

3.เลือกรางวัลที่ต้องการ กดเพื่อดูรายละเอียด จากนั้นกด แลกเลย

แต้มสะสม PAKYOK Point ได้มาจากไหน ?

-เพียงสมาชิกมียอดเดิมพัน และ เข้าเล่นในระบบอย่างต่อเนื่อง ทาง PAKYOK จะทำการให้ Point อัตโนมัติ

-สะสมแต้มพิเศษ ได้ผ่านกิจกรรมที่ประกาศตาม Promotion ในช่วงเวลานั้นๆ

​​​​​​​​​​​​​​​​เงื่อนไขแต้ม PAKYOK Point

1.จำกัดการให้แต้ม PAKYOK Point สูงสุด ไม่เกิน 100 Point ต่อคนต่อวัน

2.กิจกรรม Pakyok Point x2 จำกัดการให้รางวัลเฉพาะสมาชิกที่มียอดเคลื่อนไหวตามที่ระบบกำหนดเท่านั้น

3.กรณีที่ระบบมีการขอคืนแต้ม เปลี่ยนแปลง หรือยกเลิก ไม่ว่ากรณีใดๆ เนื่องจาก ตรวจสอบพบว่ารายการดังกล่าวผิดเงื่อนไขที่กำหนดทางเราจะหักแต้มสะสมโดยไม่ต้องแจ้งล่วงหน้า

4.PAKYOK Point ใช้สิทธิได้เฉพาะUser นั้นๆ และไม่สามารถนำแต้มสะสมจาก User อื่น มารวมได้

5.ไม่สามารถโอน PAKYOK Point ให้User อื่นได้

6.PAKYOK Point จะมีอายุภายในปีนั้นๆ นับจากปีที่ได้รับ

7.ในกรณีจัดสะสมแต้มพิเศษ ตามกิจกรรม Promotion จะมีการกำหนดวันหมดอายุPAKYOK Point โดยจะแจ้งให้ทราบในเงื่อนไข

8.PAKYOK Point ไม่สามารถ แลกเปลี่ยน หรือ ทอนเป็นเงินสดได้

9.แต้มสะสม PAKYOK Point ที่ใช้แลกรับของรางวัลแล้วไม่สามารถขอคืนได้ไม่ว่ากรณีใดๆ ทั้งสิ้น

10.PAKYOK ขอสงวนสิทธิ์ในการเปลี่ยนแปลงเงื่อนไขต่างๆ รวมทั้งสิทธิประโยชน์ โดยมิต้องแจ้งให้ทราบล่วงหน้า

11.หากมีข้อพิพาท คำตัดสินของPAKYOKถือเป็นที่สิ้นสุด

การ์ดนำโชค

การ์ดนำโชค คืออะไร ?

การ์ดนำโชค คือการ์ดสุ่มรางวัล โดยต้องนำ PAKYOK Point มาแลกเท่านั้น ไม่สามารถใช้เงินสดแลกได้

วิธีการเปิดการ์ดนำโชค

1.เลือกเมนู รางวัล (Reward)

2.เลือกแทบการ์ดนำโชค

3.เลือกแพ็คเกจการ์ดที่ต้องการ กดเพื่อดูรายละเอียด จากนั้นกด แลกเลย

4.จะสามารถเปิดการ์ดนำโชคได้ เมื่อมี แต้มPAKYOK Point เพียงพอ

*หมายเหตุ เมื่อเปิดการ์ดแล้ว ไม่สามารถเปลี่ยแปลง ขอคืน หรือยกเลิกได้ ไม่ว่ากรณีใดๆทั้งสิ้น

การเดิมพัน

วิธีการเดิมพัน มวยพักยก ผู้เล่นสามารถเดิมพันได้ในทุกราคาที่เปิดเดิมพัน ทำให้สามารถที่จะ ออกตัว เพื่อ บังทุน ของผู้เล่นได้ สามารถเดิมพัน มุมแดง หรือ มุมน้ำเงิน รวมถึงเดินพันเสมอได้ทุกรูปแบบ โดยจะเดิมพันได้ตอนที่มีการ เปิดราคา เท่านั้น ผู้เล่นทุกท่านสามารถกรอกยอดเดิมพันที่ต้องการเล่นในช่องกรอกยอดเดินพันไว้ก่อนได้

การ์ดรายการแข่งขัน

หมายเลข 1 : ชื่อรายการแข่งขัน

หมายเลข 2 : รูปภาพ และ ชื่อนักชกมุมแดง

หมายเลข 3 : รูปภาพ และ ชื่อนักชกมุมน้ำเงิน

หมายเลข 4 : ราคา ต่อรอง สำหรับมุมแดง เมื่อผู้เล่นต้องการเดิมพัน

หมายเลข 5 : ราคา ต่อรอง สำหรับน้ำเงิน เมื่อผู้เล่นต้องการเดิมพัน

หมายเลข 6 : เวลาเริ่มการแข่งขัน

หมายเลข 7 : ยก (รอบชก) ที่กำลังเปิดให้เดิมพัน

หมายเลข 8 : ปุ่มเข้าเล่น เพื่อไปยังหน้าต่างเดิมพันของการแข่งขันนั้น ๆ

หมายเลข 9 : ปุ่มสถิติ สามารถดูประวัติการแข่งขัน และรายละเอียดทางกายภาพ ของนักชกทั้ง 2 ท่าน เพื่อใช้พิจารนาในการเดิมพัน

รูปภาพ : หน้าต่างการเดิมพันมวยพักยก

วิธีการเดิมพัน มวยพักยก

1. ดูราคาต่อรองของนักชกทั้ง 2 ฝั่ง

2. กรอกยอดที่ต้องการเดิมพัน

3. เลือกฝั่งนักชกที่ต้องการเดิมพัน หรือ เดินพันเสมอ

4. เช็ครายละเอียดการเดิมพัน และ ยืนยันการเดิมพัน

5. เช็ครายการเดิมพันของท่าน ในหน้าต่างแชทการเดิมพัน

ประวัติการเดิมพัน

ผู้เล่นสามารถตรวจสอบรายละเอียดการเดิมพันย้อนหลังได้ใน ประวัติเดิมพัน โดยจะแสดงรายละเอียดทุกแมตช์ ทุกราคา ที่ผู้เล่นได้เดิมพัน รวมถึงยอดเดิพัน และผลได้เสียของแต่ละการเดิมพันด้วยเช่นกัน

รูปภาพ : ทางเข้าหน้าต่างประวัติเดิมพัน

รูปภาพ : หน้าต่างประวัติการเดิมพัน

หมายเลข 1 : ชื่อคู่ชก ที่ผู้เล่นได้เดิมพัน

หมายเลข 2 : ยอดการเดิมพันรวมของแต่ละคู่ชก

หมายเลข 3 : ฝั่งที่ผู้เล่นได้เดิมพัน

หมายเลข 4 : ราคา ต่อรอง ที่ได้เดิมพัน

หมายเลข 5 : ยก (รอบชก) และเวลาที่เดิมพัน

หมายเลข 6 : ยอดการเดิมพันในแต่ละอัตราการต่อรอง

หมายเลข 7 : ยอดสรุปผลได้เสีย
